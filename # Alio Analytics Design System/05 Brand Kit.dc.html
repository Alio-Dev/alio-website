Variables & modes
One collection "Tokens" with two modes — Light and Dark. Primitives (scales 50–950) feed semantic aliases (bg/surface/text/border/brand). Components bind only to semantic tokens, never primitives.
bg/default#FFF · #0E1017
text/primary#12141D · #F7F8FA
brand/solid#2B3990 · #6B7CD5
Naming, Auto Layout & publishing
Naming: Category/Component/Variant — e.g. Button/Primary/md. Props for state, size, icon slots.
Auto Layout: everything auto-laid; 8-pt spacing values only; min-height not fixed height; slots via nested instances.
Publishing: changes land in a "RC" branch → design review → publish with changelog entry. Breaking changes get a migration note. Version tags match code releases (v1.x).
