# Official third-party brand assets

Some marks are trademarked vendor logos that must be the vendor's **own
official SVG** — they cannot be generated, traced, or hand-authored, and were
removed from open icon sets (lucide, simple-icons) for trademark reasons.

Until each file below is added, the site shows an accessible fallback
(a text badge / monogram), so nothing breaks. Drop the official file at the
exact path to activate it. Registry: `src/config/brand-assets.ts`.

| Place file at | Used in | Get the official asset from |
|---|---|---|
| `public/brand/social/linkedin.svg` | Footer social links | brand.linkedin.com → Downloads |
| `public/brand/partners/aws.svg` | Footer "Our Partners" | AWS brand / co-marketing kit |
| `public/brand/partners/microsoft.svg` | Footer "Our Partners" | Microsoft brand assets / trademark page |

Prefer white/monochrome variants where available (the footer is on a dark
indigo background). SVG is preferred; the components size by height.
